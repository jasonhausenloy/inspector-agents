# archive

Old single-file walkthroughs and the cached demo outputs that fed them. Superseded by the live blog at https://jason.ml/inspector and the editable source at `inspector-blog.html` in the repo root (with Jasmine's `edit_server.py` for in-place editing).

Kept here for reference rather than deleted.

- `walkthrough.html` — first interactive single-file demo
- `walkthrough_redteam.html` — earlier red-team adversarial walkthrough
- `server.py` — tiny stdlib HTTP server that served `walkthrough.html` plus the JSON outputs below
- `demo_results.json`, `real_laptop_*.json`, `alibaba_*.json` — pre-computed verdicts that the old walkthrough loaded
